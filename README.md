# Route Planner Plugin

The Route Planner Plugin is a Unreal Engine plugin for calculating safe aircraft routes between a start point and a target point.

Instead of drawing a simple straight line, the plugin checks whether a route is actually flyable. It can take terrain height, minimum terrain clearance, maximum altitude, climb rate, descent rate, turn radius, and restricted or costly flight zones into account.

The goal of the plugin is to provide a clear and extendable route planning system for aircraft-aware navigation inside Unreal Engine.
